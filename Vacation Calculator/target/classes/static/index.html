Калькулятор отпускных

Проект от Ибрагимовой Ларисы Альбертовны, разработанный в рамках задания Учебного центра на направление Java Development. Микросервис на Spring Boot + Java 17 для точного расчета отпускных выплат.

1 GET "/api/calculate"
2 // Параметры:
3 • averageSalary: double — средняя зарплата за 12 месяцев (₽)
4 • vacationDays: int — количество календарных дней отпуска
5 • startDate: String (опционально) — дата начала отпуска для учета праздников
6 // Ответ:
7 { "vacationPay": 9569.96, "actualWorkingDays": 14 }

Реализация проекта

Микросервис разработан в соответствии со всеми требованиями технического задания на чистоту кода, архитектурную целостность и покрытие тестами.

Чистая архитектура

Соблюдены принципы SOLID, использован паттерн проектирования Strategy для расчета с/без учета праздников, слоистая архитектура: Controller → Service → Model → Util.

Учет праздников РФ

При указании даты начала отпуска автоматически исключаются нерабочие праздничные дни (1–8 января, 23 февраля, 8 марта и др.) из расчетного периода в соответствии с ТК РФ.

Покрытие тестами

Полное покрытие unit-тестами бизнес-логики: базовый расчет, учет праздников в январе/феврале, граничные случаи (1 день отпуска), валидация параметров.
